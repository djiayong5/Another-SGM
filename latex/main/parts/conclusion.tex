В работе получены следующие научные результаты:

\begin{enumerate}
\item Из всех элементов, существующих в диаграммах состояний UML, выделены основные, базовые элементы.
  Это было сделано для упрощения модели и облегчения построения формальной семантики.
\item Построена формальная семантика для выделенного подмножества элементов диаграмм состояний UML,
  Введены три класса систем, по степени их детерминированности.
\item Были исследованы существующие паттерны проектирования, используемые для построения реактивных
  систем, разработан новый паттерн проектирования O-State, помогающий реализовать построенную семантику.
\item Разработана библиотека O-State на языке Java, содержащая классы и интерфейсы, необходимые для
  реализации предложенного паттерна проектирования.
\item Результаты работы внедрены в ядре Taiga системы PCMS4.
\end{enumerate}

Возможны следующие пути развития работы:

\begin{enumerate}
\item Расширение семантики для поддержки многопоточной обработки событий.
\item Модификации и расширения библиотеки O-State для более удобного использования.
\item Рассмотреть варианты использования предложенной семантики для не объектно-ориентированных языков.
\end{enumerate}
